PCL 的下列自定义项中支持使用替换标记：
<ul>
<li>
<code>设置 → 启动</code> 与 <code>版本设置 → 设置</code>
<ul>
<li><code>启动选项 → 游戏窗口标题、自定义信息</code></li>
<li><code>高级选项 → Java 虚拟机参数、游戏参数、启动前执行命令</code></li>
</ul>
</li>
<li>
<code>设置 → 个性化 → 主页</code>
<ul>
<li>联网更新的下载地址</li>
<li>主页文件的 XAML</li>
</ul>
</li>
<li>自定义帮助文件的 JSON 与 XAML</li>
</ul>

你可以在自定义主页的 URL 等地方使用替换标记，以根据不同的参数返回不同的 XAML。

替换标记会按照本页面中的顺序从上往下依次替换。因此若替换后的内容也包含替换标记，在顺序恰当的时候会出现嵌套替换。

## 基础信息

| 替换标记 | 简介 | 详情 | 示例 |

| `{pcl_version_branch}` | PCL 版本分支 | 快照版为 `Snapshot`，正式版为 `Release`，开发版为 `Debug` | `Release` |
| `{pcl_branch}` | PCL 分支名 | 官方版为 `Official`，开源版为 `OpenSource`，社区版本可能有其他值 | `Official` |
| `{identify}` | 识别码 | 每个用户均不一样的识别码 | `B49F-F9F4-2D63-2FE3` |
| `{path}` | PCL 所在文件夹 | PCL 程序所在文件夹的完整路径，以 `\` 结尾 | `D:\Minecraft\程序\` |
| `{path_with_name}` | PCL 程序路径 | PCL 程序的完整路径，包含文件名 | `D:\Minecraft\程序\PCL.exe` |
| `{path_temp}` | 缓存文件夹路径 | PCL 所使用的缓存文件夹路径，可在设置中修改，以 `\` 结尾 | `C:\Users\...\Temp\PCL\` |

这些替换标记中的每一个都会被独立处理，可能出现不一样的结果。

| 替换标记 | 简介 | 详情 | 示例 |
| --- | --- | --- | --- |
| `{hint}` | 随机 “你知道吗？” 条目 | 随机替换为一条启动游戏时显示的 “你知道吗？” 条目若 PCL 文件夹中有 `hints.txt`，则为其中的一行随机内容 | `在第一次启动游戏时，PCL 会自动将语言设置为中文。` |
| `{cave}` | 随机回声洞条目 | 随机替换为一条 `更多 → 百宝箱 → 回声洞` 中的条目 | `歪比巴卜？` |
| `{setup:设置名}` | 读取 PCL 对应名称的设置 | 你可以在 [ModSetup.vb](https://github.com/Meloong-Git/PCL/blob/main/Plain%20Craft%20Launcher%202/Pages/PageSetup/ModSetup.vb) 的开头查看其列表若上述文件对应行有 `Encoded:=True`，则该项设置已加密，不允许读取若上述文件对应行有 `Source:=SetupSource.Version`，则该项设置是版本独立设置，本替换标记会读取当前选中版本的对应设置 | `{setup:UiLauncherTransparent}` → `400` |
| `{variable:变量名}` | 读取使用自定义事件设置的变量 | 读取此前使用 [修改变量 / 写入变量](https://github.com/Meloong-Git/PCL/wiki/%E8%87%AA%E5%AE%9A%E4%B9%89%E4%BA%8B%E4%BB%B6#%E4%BF%AE%E6%94%B9%E5%8F%98%E9%87%8F--%E5%86%99%E5%85%A5%E5%8F%98%E9%87%8F) 自定义事件设置的变量如果变量未设置过，则替换为空 | `{variable:storage}` → `test_value` |
| `{variable:变量名:默认值}` | 读取使用自定义事件设置的变量 | 与上方的一样，不过如果变量未设置过，则替换为默认值 | `{variable:dont_exist:default}` → `default` |
